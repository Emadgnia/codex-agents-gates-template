# Deploy Runbook

## 1. Pre-Deploy
- Confirm approved plan and release scope.
- Confirm artifact version and target environment.
- Confirm secrets and runtime config.

## 2. Deploy
- Execute deployment command/tooling for target environment.
- Monitor logs during rollout.

## 3. Verify
- Run smoke checks.
- Validate key user flows and integrations.

## 4. Report
- Record deployment time, version, and status.
- Record incidents or rollback actions.
