# Security Policy

## Supported Scope
This repository is a template pack (documentation + process artifacts). Security reports should focus on:

- Secret exposure in templates or examples
- Unsafe operational guidance
- Risky defaults that could lead to misuse in downstream projects

## Reporting a Vulnerability
Please do not open public issues for sensitive findings.

Use GitHub Security Advisories for private reporting, or contact the maintainer directly through a private channel.

## Response Targets
- Initial acknowledgement: within 3 business days
- Triage decision: within 7 business days
- Remediation timeline: based on severity and impact
